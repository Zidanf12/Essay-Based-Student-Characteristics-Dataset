# Essay-Based Student Characteristics Dataset

## Description

This dataset contains student essay texts used to identify student characteristics based on learning styles and personality types.

The dataset was developed for research on student characteristic identification using machine learning methods.

## Dataset Structure

The dataset contains the following columns:

- `ID`: Anonymous identifier for each data sample.
- `Essay`: Student essay text.
- `Learning_Style`: Student learning style based on the VARK model.
- `Personality`: Student personality type, categorized as Extrovert or Introvert.

## Learning Style Categories

The learning style labels consist of:

- Visual
- Auditory
- Read/Write
- Kinesthetic

## Personality Categories

The personality labels consist of:

- Extrovert
- Introvert

## Data Processing

The dataset may include data that has undergone preprocessing, balancing, and text augmentation to support machine learning classification.

## Usage

This dataset can be used for academic and research purposes, particularly for text classification, learning style prediction, personality classification, and machine learning research.
